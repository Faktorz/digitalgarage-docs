Overview

Events in Digital Garage are modeled based on events that happen to API objects in an Digital Garage cluster. Events allow Digital Garage to record information about real-world events in a resource-agnostic manner. They also allow developers and administrators to consume information about system components in a unified way.

Viewing Events with the CLI

You can get a list of events in a given project using the following command:

$ oc get events [-n <project>]

Viewing Events in the Console

You can see events in your project from the web console from the Browse → Events page. Many other objects, such as pods and deployments, have their own Events tab as well, which shows events related to that object.
