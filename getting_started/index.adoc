[.lead]
{product-title} is based on the following technologies an service providers:
- Openshift Origin version 3, Red Hat's container development platform that makes it easy for developers to quickly build and launch,
and scale container-based web apps in a public cloud environment.
- Kubernetes, AUTOMATED DEPLOYMENT, SCALING AND APPLICATION HEALTH. Created by Google, Kubernetes is an open-source system for automating deployment, operations and scaling of containerized applications.
- Docker Containers, TINY, LIGHTER AND FASTER THAN VIRTUAL MACHINES. We use Docker to encapsulate and run applications, bringing more agility and consistency when deploying applications.
- Amazon Web Services, WORLD LEADER IN CLOUD INFRASTRUCTURE.

Check out the following topics to get started as an *application developer*
trying out {product-title}:

* Step through a xref:../getting_started/basic_walkthrough.adoc#getting-started-basic-walkthrough[*basic walkthrough*]
using the web console and create your first project and application.
* Go xref:../getting_started/beyond_the_basics.adoc#getting-started-beyond-the-basics[*beyond the basics*] and get
hands-on with the CLI.
endif::digital-garage,faktorz[]
